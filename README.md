# 🧠 HIX: neovim-ollama

**Connect your local LLMs directly within Neovim.**

---

## 🚧 Work in Progress

I'm actively developing this plugin to seamlessly integrate local LLMs into Neovim. While it's in the early stages, the goal is to provide a lightweight, efficient interface for interacting with models like Ollama, enhancing your coding and writing experience.

---

## ⚙️ Features (Planned)

- **Local LLM Integration**: Connect to your local Ollama server within Neovim.
- **Model Selection**: Choose from available models for various tasks.
- **Prompt Contextualization**: Send selected buffer content as context to the model.
- **Custom Actions**: Define how to handle and display model responses.
- **Configuration Flexibility**: Easily adjust settings to fit your workflow.
